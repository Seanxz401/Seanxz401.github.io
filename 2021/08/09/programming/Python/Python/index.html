代码规范

PEP8

代码布局

  • 缩进：4个空格；
  • 空格选择：首选用空格缩进，不用tab；
  • 每行代码的最大长度：<=79个字符；
  • 空白行使用规则
    • 一个空白行—函数与函数之间&类与类之间；
    • 两个空白行分隔顶层函数和类定义；
  • 原文件编码选择：Python3为UTF-8，2为ASCII
  • 导入库函数规则
    • 一行一个import；
    • 位置：顶部，模块注释和文档字符串之后，模块全局变量和常量之前；
    • 顺序：标准库→第三方库→本地应用程序/库特定；
    • 从包含类的模块中导入一个类是，避免通配符（*）导入，避免过多的内存占用。

    • 逗号，分号和冒号前面

字符串

如需单引号字符，则用双引号包含单引号；若既需要单引号又需要双引号，可使用转义字符\。

特殊字符：再加一个转义字符

使用r可以让反斜杠不发生转义。

字符串内部换行：'''...'''

索引：1.与一般数组相似从0开始；2.以-1为末尾的开始位置，从后往前以此-2，-3……

截取：

  • str[0:-1]所有字符
  • str[2:5]截取第三个到第五个字符
  • str[2:]截取从第三个往后所有字符

加号+可连接字符串，*可复制（重复输出：str*2）

python中的字符串本身不可改变。

列表

List ：list=['1','2']方括号，逗号

截取：与字符串类似，以及+*

修改：使用赋值语句可进行重新赋值即修改。

str[0::2]从第一个元素开始，每两个元素（2表示步长）取一个元素，即取出所有偶数索引值的元素。

内置方法：

  • append()在list尾部进行添加；
  • pop()删除并在控制台显示list的最后一个元素；

嵌套：list=[1,[2,3]]，list中可包含所有数据类型，包括本身list

元组

tuple=('1','2')小括号

与list类似，但元组中的元素不能修改，但是元组中可包含可变的对象。

特殊情况：只有一个元素时，仍要在元素后添加逗号。

string，list，tuple都属于sequence。

字典

dictionary，是无序的对象集合，通过键来存取。不支持截取和索引。

dictionary={'key':'value','name':'test'}大括号，键值对。

键必须为不可变类型，且唯一。重复的会被覆盖。

集合

set。用大括号{}和set()创建集合，但只能set()创建空集合。parame = {'value1','value2'}或set('value')

元素唯一，重复的会被自动去掉。

位运算符

位运算符是用来操作二进制的。涉及不多，拜拜了您嘞

https://blog.csdn.net/meng_xin_true/article/details/103548077

http://c.biancheng.net/view/2184.html

  • &：按位与运算符，操作的是数据在内存中存储的原始二进制位，两个位都为1时结果才为1，否则为0；
  • |：按位或运算符，有一个为1就行；
  • ^：按位异或运算符，两个位不同结果为1；
  • ~：按位取反运算符，单目运算符，对参与运算的二进制位取反，如~1=0；
  • <<：左移运算符，如9<<3，把9的二进制位全部左移3位，高位丢弃，低位补0；
  • >>：右移运算符，低位丢弃，高位：如果数据的最高位是 0，那么就补 0；如果最高位是 1，那么就补 1。

打开模式

  • +：打开一个文件进行更新（读写）；
  • -r：读取，默认，可省略；
  • -rb：以二进制格式打开，从头，一般用于非文本文件如图片；
  • -r+：读写，从头；
  • -w：写，原有内容会被删除，如该文件不存在会新建；
  • -w+：读写，从头，会删除原内容，可进行新建；
  • -a，a+：从文件末尾写

关键字参数

调用函数时指定参数名称，可不管参数传递顺序，若指定则所有非缺省参数都必须指定。
